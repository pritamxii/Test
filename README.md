# Habit Master Pro 2026 - Full Stack

A comprehensive Habit Tracking application built with HTML, CSS, JavaScript, and a Python (Flask) backend.

## Features
- Interactive Habit Tracking with Weighted Progress.
- Multi-language Support (English & Bengali).
- Data Persistence using Python Backend and JSON storage.
- Visual Analytics with Chart.js.

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Python (Flask)
- **Library:** Chart.js, Canvas-Confetti
